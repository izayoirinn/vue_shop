#git 基本命令
    创建新分支: git checkout -b login
    查看分支: git branch
    切换分支: git checkout 分支名
    合并分支: 在主分支执行 git merge login 表示将 login分支合并到主分支
    推送到远程仓库: git push
    第一次推送其他分支到远程仓库: git push -u origin login
